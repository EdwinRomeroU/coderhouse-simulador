Simulador JavaScript

Este es un simulador desarrollado en JavaScript que te permite adivinar una palabra, mediante el ingreso de diferentes letras. El objetivo es adivinar la palabra antes de que se acaben los intentos.

Instrucciones

Para jugar, debes adivinar la palabra ingresando letras una por una.
Tienes un número limitado de intentos (15) para adivinar la palabra completa.
En el momento que desees, digita "salir" para abandonar el juego.
